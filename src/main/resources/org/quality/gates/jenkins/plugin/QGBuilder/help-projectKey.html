• Project Key is the unique key Sonar analysis uses for your project.

  • To get the Project Key go to your Sonar instance (login if necessary) and click on the project whose key you need. You will be redirected to an URL like this: 'http://myhost.:9000/overview?id=123'. Another click on the project name will redirect you to the same view with the URL looking like this: 'http://localhost:9000/overview?id=com.company%3AMyProject'.
  • 'com.company%3AMyProject' is your Project Key
